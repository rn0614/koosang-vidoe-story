# Chrome Debugging

# Source
> 실질적으로 source를 디버깅 할 수 있는 탭

Scope : 현재 함수의 스콥프
Call Stack : 현재 함수를 부른 위치 - 비동기도 순차적으로 보여줌
Break Point : 조건부 중단점 생성 가능
XHR/fetch Breakpoints : fetch 들어가는 부분을 debugging

Group by, Authored/Deployed : 를 통행 개발/ 배포소스 보기
# Elements
